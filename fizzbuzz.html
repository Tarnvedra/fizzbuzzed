About the App

This is a mini project written using HTML5,CSS3 and JavaScript and Bootstrap 4. The app is the classic whiteboarding challenge where any number divisible by 3 is a fizz number and a number divisble by 5 is a buzz number and divisble by both is a fizzbuzz number. The second button gives a list of the fibonacci sequence of numbers
